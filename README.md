# Text Analytics with SAS® Model Studio  

In this Hands-on Workshop, you will learn and practice the essentials to get started with SAS®  Visual Text Analytics in Model Studio. You will learn to create a project; access a corpus of text documents; extract concepts, terms, and topics; and interpret your findings. Attendees are provided with access to SAS®  Visual Text Analytics software and step-by-step instructions to get started.
